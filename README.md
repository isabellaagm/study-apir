
# Study APIr
API Java para estudo de Spring boot

## Instalação

- Limpar e criar a pasta _/target_
```
mvn clean package
```


## Navegação

### Executar a API
-   Executando o __Maven__

```   
mvn spring-boot:run
```

### Documentação da API (Swagger)
- http://localhost:8080/swagger-ui.html
